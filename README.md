# Hardhat Smartcontract Lottery (Raffle)

## Quickstart

```
git clone https://github.com/worksofallwin/hardhat-raffle.git
cd hardhat-raffle
yarn
```

# Usage

Deploy:

```
yarn hardhat deploy
```

## Testing

```
yarn hardhat test
```

### Test Coverage

```
yarn hardhat coverage
```
